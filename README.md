# StockAnalyser

This is a python based CLI stock analyser built without any GUI, focusing on core programming concepts of python with optional yfinance integration. Includes features like viewing stocks, deep dive analysis, sector comparison, portfolio tracking, research report generation, custom stock entry, live price updates, and CSV export. 

This project started as a way to better understand how basic stock market analysis works while also strengthening my Python fundamentals. Instead of relying on external tools or complex libraries, the idea was to build everything from scratch using simple logic and standard modules, and then gradually add more practical features. The stock data used in the program is structured in a way that makes it easy to extend, modify, or plug into real-time sources like yfinance when needed. The goal wasn’t to create something overly advanced, but something that feels usable and realistic for everyday learning and experimentation.

Each feature is designed to reflect a small but important part of how investors typically evaluate stocks. For example, the deep dive section focuses on price, EPS, P/E ratio, and 52-week range to give a quick snapshot of valuation. The sector comparison helps in understanding how different industries stack up against each other, while the portfolio manager introduces basic concepts like tracking invested value, current value, and returns. The research report feature was added to simulate how a simple equity note might look, including a recommendation based on valuation logic.

I also tried to keep the interface clean and readable despite being terminal-based, using formatted output and small visual elements like progress bars. Another focus was making the program interactive, so users can add their own stocks, update data, and export results for further analysis outside the script. Overall, it can serve as a starting point for anyone interested in combining finance concepts with programming.
